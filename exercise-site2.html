1.article要素

article要素は、独立した1つのコンテンツをマークアップする要素である。

あるサイトの記事や投稿されたコメントなど、既に完成されたものを同じように表現する際に用いると適切であるということが分かった。

2.blockquote要素・cite属性

blockquote要素は記述される文章が、外部からの引用・転載されたものである場合によく用いられる。

また合わせてcite属性を付与することで引用元のURLを添えることが出来る。

HTML の <blockquote> 要素 (HTML ブロック引用要素) は、内包する要素の文字列が引用文であることを示します。
通常、字下げを伴ってレンダリングされます (整形方法については注意の項を参照してください)。
cite属性により引用元の文書の URL を、 <cite> 要素により引用元の文書のタイトルなどを明示可能です。

MDN Web Docs <blockquote>:ブロック引用要素

上記の様に引用文であるということがわかりやすいようにする必要がある。

cite属性を付与していない場合、「無断転載」とみなされる恐れがあるため引用元にアドレスを書くことはほぼ必須であるということを理解した。

3.dl,dt,dd要素

dl要素は説明リスト、dtはリスト内に存在する用語、ddはその用語の説明を示す。

ul,ol要素を用いたリストの違いは、リストに説明が必要であるかそうでないかにある。用語の説明をしたい時にはdl要素を用いる。

じゃんけん
グー
・チョキに勝ちパーに負ける
チョキ
・パーに勝ちグーに負ける
パー
・グーに勝ちチョキに負ける

dt要素内（グー、チョキ、パー）はstrong要素を用いて太字で描画している。

4.figure,figcaption要素

figure要素は写真、図表、コードなどを扱うことができ表示させることが出来る。

figcaption要素はfigure要素で用いているコンテンツを説明するために使う。

figcaption内でもcite属性を付与することで引用元のURLを添えることが出来る。

この場合figcaption要素は記述した内容が表示されるため引用URLも表示される。
